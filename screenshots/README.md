# Screenshots

This directory contains screenshots of the Vyākṛti IDE in action.

## How to Capture

1. Start the IDE (see README.md → Quick Start)
2. Open a `.vya` file in the editor
3. Use your OS screenshot tool or browser dev tools (F12 → device toolbar → 1920×1080)
4. Save images here as PNG files

## Suggested Captures

| File | Description |
|------|-------------|
| `ide-editor.png` | Editor panel with Vyākṛti syntax highlighting |
| `ide-compile-output.png` | Compile output showing AST, bytecode, and diagnostics |
| `ide-repl.png` | WebSocket REPL session in the terminal panel |
| `ide-settings.png` | Settings panel with theme toggle |
